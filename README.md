# 下载与安装

> git clone 到本地

## Build Setup

``` bash
# git clone
git clone https://github.com/srboyy/nodeVueDemo.git

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# //启动服务器
cd server 

# //启动服务器
node app.js 

# 显示如下
    console.log('--------------');
    console.log('数据库登录成功-');
    console.log('--------------');

# 打开浏览器
localhost:8080



```
